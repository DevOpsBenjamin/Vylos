Release Notes — @vylos/core + @vylos/cli

========================================
v1.2.0
========================================

BREAKING CHANGES:
- VylosConfig.languages changed from `string[]` to `LanguageEntry[]`
  Before: languages: ['en', 'fr']
  After:  languages: [{ code: 'en', label: 'English' }, { code: 'fr', label: 'Français' }]
  Migration: Update vylos.config.ts in all projects.

NEW:
- LanguageEntry type exported from @vylos/core ({ code: string, label: string })
- SettingsMenu language selector now reads labels from config (no more hardcoded English-only)

FIXES:
- SettingsMenu: language dropdown was hardcoded to English only, now dynamically populated from config.languages

INTERNAL:
- Template vylos.config.ts updated to use LanguageEntry format
- All sample projects (1-basic, 2-advanced, 3-phone) updated

NOTE: External consumer projects (e.g. Mina Alchemist) must update their vylos.config.ts
